  <action>
Create `voice-agent/app/services/calendar.py` with the following public API:

**CalendarError** — custom exception for all Google Calendar failures. Message should be caller-safe (no raw API stack traces).

**get_calendar_service(db, client_id) -> Resource**
- Query `oauth_tokens` for the given `client_id` (select latest by id desc, limit 1)
- Decrypt `encrypted_access_token` and `encrypted_refresh_token` via `decrypt_token()`
- Build `google.oauth2.credentials.Credentials` object with the decrypted tokens, `token_expiry`, and the Google client_id/secret from `get_settings()`
- If `credentials.expired and credentials.refresh_token`: call `credentials.refresh(google.auth.transport.requests.Request())` then persist updated token back to DB (UPDATE `oauth_tokens` SET encrypted_access_token=…, token_expiry=… WHERE id=…)
- Build and return `googleapiclient.discovery.build("calendar", "v3", credentials=credentials)`
- On any exception: raise CalendarError with a safe message

**get_free_slots(db, client_id, window_start, window_end, max_slots=3) -> list[datetime]**
- Load `Client` row for `client_id` to get `working_days`, `business_hours`, `slot_duration_minutes`, `buffer_minutes`, `lead_time_minutes`, `timezone`
- Parse `business_hours` dict: `start` and `end` as "HH:MM" strings
- Enforce `lead_time_minutes`: earliest searchable start = `now(tz) + timedelta(minutes=lead_time_minutes)`
- Clamp `window_start` to max(window_start, earliest start)
- Call Google Calendar `freebusy` API: `service.freebusy().query(body={...}).execute()` for the calendar "primary" in the window
- Iterate candidate slots: for each day in [window_start.date() .. window_end.date()]:
  - Skip if `day.isoweekday()` not in `working_days`
  - Iterate slot starts from business_hours start to end in increments of `slot_duration_minutes + buffer_minutes`
  - Skip slot if it falls before clamped window_start
  - Skip slot if it overlaps any busy period from freebusy response
  - Append slot datetime (timezone-aware, in client timezone) to results
  - Stop when `len(results) == max_slots`
- Return list (may be empty if no free slots in window)
- On Google API error: raise CalendarError

**create_event(db, client_id, start_dt, end_dt, summary, description) -> str**
- Build event dict: summary, description, start={dateTime, timeZone}, end={dateTime, timeZone}
- Call `service.events().insert(calendarId="primary", body=event).execute()`
- Return `event["id"]`
- On error: raise CalendarError

**update_event(db, client_id, event_id, new_start_dt, new_end_dt) -> str**
- Call `service.events().patch(calendarId="primary", eventId=event_id, body={start:…, end:…}).execute()`
- Return `event["id"]`
- On error: raise CalendarError

**delete_event(db, client_id, event_id) -> None**
- Call `service.events().delete(calendarId="primary", eventId=event_id).execute()`
- On error: raise CalendarError

Implementation notes:
- All datetime parameters are timezone-aware (use `zoneinfo.ZoneInfo(client.timezone)`)
- `get_calendar_service` is an internal helper called by all public functions — do not expose as public API
- Use `logging.getLogger(__name__)` for all log statements; no `print()` calls
- Keep each function under 50 lines
- Export via `__all__`: `["CalendarError", "get_free_slots", "create_event", "update_event", "delete_event"]`
