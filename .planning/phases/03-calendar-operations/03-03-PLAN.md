Create `voice-agent/app/services/appointment.py` with the following public API:

**AppointmentError** — custom exception. Message is caller-safe. Wraps CalendarError internally.

**BookingRequest** — frozen dataclass holding the 6 required fields:
```python
@dataclass(frozen=True)
class BookingRequest:
    name: str
    phone: str
    email: str
    address: str
    problem_description: str
    access_notes: str
```

**BookingResult** — frozen dataclass:
```python
@dataclass(frozen=True)
class BookingResult:
    confirmed: bool          # True = booked, False = slot busy (alternatives offered)
    event_id: str | None     # Set when confirmed=True
    slot: datetime | None    # Confirmed slot (when confirmed=True)
    alternatives: list[datetime]  # Populated when confirmed=False (up to 2)
```

**book_appointment(db, client_id, requested_slot, booking_request) -> BookingResult**
- Call `get_free_slots(db, client_id, requested_slot, requested_slot + timedelta(minutes=1), max_slots=1)` to check if the exact slot is free
- If free: call `create_event()` with summary=booking_request.name, description built from all 6 fields (name, phone, email, address, problem, access notes — newline-separated); return BookingResult(confirmed=True, event_id=..., slot=requested_slot, alternatives=[])
- If busy: call `get_free_slots(db, client_id, requested_slot, requested_slot + timedelta(days=30), max_slots=2)` for alternatives; return BookingResult(confirmed=False, event_id=None, slot=None, alternatives=alternatives)
- On CalendarError: raise AppointmentError with "I'm having trouble accessing the calendar right now"

**find_slot_in_window(db, client_id, window_start, window_end) -> list[datetime]**
- Window is capped at 30 days: `window_end = min(window_end, window_start + timedelta(days=30))`
- Calls `get_free_slots(db, client_id, window_start, window_end, max_slots=3)`
- Returns list (empty = no slots found → caller should transfer)
- On CalendarError: raise AppointmentError

**AppointmentMatch** — frozen dataclass:
```python
@dataclass(frozen=True)
class AppointmentMatch:
    event_id: str
    summary: str
    start: datetime
    end: datetime
```

**find_appointment(db, client_id, caller_name, appointment_date) -> list[AppointmentMatch]**
- Build search window: appointment_date 00:00 to 23:59 in client timezone
- Call Google Calendar `service.events().list(calendarId="primary", timeMin=..., timeMax=..., q=caller_name).execute()`
- Filter results: keep only events where caller_name appears in event["summary"] (case-insensitive)
- Return list of AppointmentMatch objects (may be empty or multiple)
- On CalendarError: raise AppointmentError

Note: `find_appointment` needs `get_calendar_service` internally. Import it from `app.services.calendar` as a private import (`from app.services.calendar import get_calendar_service as _get_calendar_service`).

**reschedule_appointment(db, client_id, event_id, new_start_dt, slot_duration_minutes) -> str**
- Compute new_end_dt = new_start_dt + timedelta(minutes=slot_duration_minutes)
- Call `update_event(db, client_id, event_id, new_start_dt, new_end_dt)`
- Return event_id
- On CalendarError: raise AppointmentError

**cancel_appointment(db, client_id, event_id) -> None**
- Call `delete_event(db, client_id, event_id)`
- On CalendarError: raise AppointmentError

Implementation rules:
- All functions are `async`
- Use `logging.getLogger(__name__)` — no print()
- Keep each function under 50 lines
