Write voice-agent/app/services/ingestion.py implementing:

**Constants (module-level):**
- MAX_FILE_BYTES = 10 * 1024 * 1024  (10MB)
- CHUNK_TOKENS = 400
- CHUNK_OVERLAP_TOKENS = 80
- CSV_ROWS_PER_CHUNK = 10
- EMBED_MODEL = "text-embedding-3-small"
- EMBED_BATCH_SIZE = 100  (OpenAI allows up to 2048 inputs per call; 100 is safe)
- SIMILARITY_THRESHOLD = 0.75  (used by RAG query service in plan 02-03; define here as the canonical value)

**parse_file(filename: str, content: bytes) -> str:**
- If len(content) > MAX_FILE_BYTES: raise ValueError(f"File exceeds 10MB limit ({len(content)} bytes)")
- Dispatch on filename extension (case-insensitive):
  - .pdf: use pypdf.PdfReader(io.BytesIO(content)); join page.extract_text() for all pages
  - .docx: use docx.Document(io.BytesIO(content)); join para.text for all paragraphs
  - .txt: content.decode("utf-8", errors="replace")
  - .csv: content.decode("utf-8", errors="replace") — returns raw CSV text for chunk_csv to handle
  - other: raise ValueError(f"Unsupported file type: {ext}")

**chunk_text(text: str) -> list[str]:**
- Use tiktoken.get_encoding("cl100k_base")
- Encode full text to tokens
- Slide window of CHUNK_TOKENS with CHUNK_OVERLAP_TOKENS step
- Decode each window back to string
- Skip empty/whitespace-only chunks
- Returns list of chunk strings

**chunk_csv(content: bytes) -> list[str]:**
- Decode as UTF-8
- Use csv.DictReader or csv.reader to iterate rows
- Group CSV_ROWS_PER_CHUNK rows into one chunk string (header + rows)
- Include the header in every chunk for context
- Returns list of chunk strings

**chunk_file(filename: str, content: bytes) -> list[str]:**
- If .csv extension: call chunk_csv(content)
- Otherwise: text = parse_file(filename, content); return chunk_text(text)

**async embed_chunks(chunks: list[str]) -> list[list[float]]:**
- Uses openai.AsyncOpenAI(api_key=get_settings().openai_api_key)
- Batches chunks in groups of EMBED_BATCH_SIZE
- Calls client.embeddings.create(model=EMBED_MODEL, input=batch) per batch
- Returns flat list of embeddings aligned with input chunks

**async delete_document(db: AsyncSession, client_id: str, doc_name: str) -> int:**
- DELETE FROM embeddings WHERE client_id = :client_id AND doc_name = :doc_name
- Use sqlalchemy delete() with where() clause
- Return number of rows deleted (result.rowcount)
- Commit the transaction

**async ingest_document(db: AsyncSession, client_id: str, filename: str, content: bytes) -> dict:**
- Call chunk_file(filename, content) to get chunks
- If len(chunks) == 0: raise ValueError("Document produced no content after parsing")
- Call embed_chunks(chunks) to get vectors
- Call delete_document(db, client_id, filename) to remove any prior version
- Bulk insert Embedding rows: for i, (chunk, vector) in enumerate(zip(chunks, vectors)):
    db.add(Embedding(client_id=client_id, doc_name=filename, chunk_index=i, content=chunk, embedding=vector))
- await db.commit()
- Return {"doc_name": filename, "chunks_ingested": len(chunks)}

All functions must have type annotations. Use logging (not print) for any debug output.
