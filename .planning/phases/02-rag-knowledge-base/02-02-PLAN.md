**Create voice-agent/admin/templates/documents.html:**
Simple, functional HTML page (no CSS framework required — plain HTML is fine):
- Page title: "Knowledge Base — {client_id}"
- A form with:
  - Hidden field: client_id (from query param)
  - File input (accept=".pdf,.docx,.txt,.csv")
  - Submit button: "Upload"
  - Form action: POST /admin/documents/upload enctype="multipart/form-data"
- A flash message area (show `message` from template context if present, styled with color based on `error` boolean)
- An unordered list of existing documents: each item shows doc_name + chunk count + a Delete button
  - Delete button: small form with POST /admin/documents/delete, hidden fields for client_id and doc_name
- If no documents: show "No documents ingested yet."
